# Laboratorio Poético Brutalista 🖋️💥

Una herramienta interactiva de creación lírica con una estética de alto impacto. Diseñada para romper el bloqueo creativo mediante ejercicios guiados y generadores aleatorios.

## Características

-   **Módulo Haiku**: Escritura guiada con contador de sílabas en tiempo real (métrica 5-7-5).
-   **Módulo Acróstico**: Guía visual vertical para la escritura de versos basados en una palabra clave.
-   **Generador de Chispas**: Crea combinaciones inesperadas de sustantivos y adjetivos para disparar la imaginación.
-   **Blackout Poetry**: Sistema de reescritura que permite "oscurecer" textos clásicos para descubrir nuevos significados.
-   **Antología Local**: Guarda tus creaciones durante la sesión y descárgalas en formato `.txt`.
-   **Diseño Brutalista**: Interfaz agresiva y honesta con sombras rígidas, bordes gruesos y tipografía Serif/Mono.
-   **Modo Zen (Noche/Día)**: Ajuste de contraste para facilitar la escritura en cualquier ambiente.

## Tecnologías

-   **React + Vite**
-   **Tailwind CSS** (Estilo Brutalista personalizado)
-   **Lucide React** (Iconografía)

## Despliegue en Vercel

1.  Sube este código a un repositorio de GitHub.
2.  Conecta tu cuenta de GitHub a [Vercel](https://vercel.com).
3.  Importa el proyecto y haz clic en "Deploy".
